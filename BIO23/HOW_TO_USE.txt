Instructions for Running the Automation Script and Scripts in the bio23 Project:

Before you begin:
Make sure you have installed the required libraries and modules for the bio23 project:

- Python3:
Depending on your operating system, Python3 may already be installed. If not, visit the official Python website (https://www.python.org/) and follow the installation instructions for your OS.

- tkinter:
sudo apt-get install python3-tk

- BLAST:
https://blast.ncbi.nlm.nih.gov/Blast.cgi

- MAFFT:
sudo apt-get install mafft

- Clustal Omega:
sudo apt-get install clustalo

- Muscle:
sudo apt-get install muscle

- phyml:
sudo apt-get install phyml

- FastTree:
sudo apt-get install fasttree


Now, follow the step-by-step instructions for running each script and the graphical user interface (GUI) in the bio23 project:

1. GUI:
a. Navigate to the project directory <bio23>.
b. Open the terminal.
c. Execute the following command: python3 gui.py

2. Database Retrieval from NCBI:
a. Go to the project directory <bio23/NCBI>.
b. Open the terminal.
c. Run the following command: ./NCBI/ncbi_script.sh <db-name> <accession-number>

3. Sequence Alignment:
a. Change to the project directory <bio23/SA>.
b. Open the terminal.
c. For global sequence alignment, run: ./SA/global.sh -s <saQu_filePath> -q <saRef_filePath> -o <sa_output_file_name>
d. For local sequence alignment, run: ./SA/local.sh -d <saRef_filePath> -q <saQu_filePath> -o <sa_output_file_name>

4. Multiple Sequence Alignment:
a. Switch to the project directory <bio23/MSA>.
b. Open the terminal.
c. To perform multiple sequence alignment, execute:
With file input: ./MSA/MSA.sh -msaTool <algorithm> -outName <output_file_name> -seqPath <msa_filePath>
With paste input: ./MSA/MSA.sh -msaTool <algorithm> -outName <output_directory> -seqPaste <<EOF EOF>

5. Phylogenetic Tree:
a. Move to the project directory <bio23/PTREE>.
b. Open the terminal.
c. Run the following command: ./PTREE/PHYLO.sh --inName <tree_filePath>

6. Full Automation:
a. Navigate to the project directory <bio23/AUTOMATION>.
b. Open the terminal.
c. Run the automation script using the command: ./AUTOMATION/auto.sh <db> <accession-number> <alignNumber> <algorithm>
